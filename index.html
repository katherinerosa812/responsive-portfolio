My Portfolio

Unplugged

Unplugged is a responsive website I created for a class project. During this project, I learned about media queries, break points and how to use Flexbox.

Rogue Pickings

This is a website I did for a class project. Working only from a PSD, I hand-coded this website from scratch. I learned how to extract the information I needed from the PSD. This project was also a great exercise in using floats.

Jubilee Austen

This is a portfolio website project I did for class. I hand-coded the website, purchased a domain name and hosting, uploaded my site files via FTP. Then, I placed the project under version control using Git, and backed-up the site files to Github.

Knotty Mess

This is a Wordpress site I created for a small e-commerce business. The client wanted to move their shop away from Etsy. I selected a theme, customized it using HTML and CSS, and set up a WooCommerce shop. I also added several plugins, including an Instagram plugin.
